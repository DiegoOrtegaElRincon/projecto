<!-- GETTING STARTED -->
## Getting Started

La instalación es muy simple

### Prerequisites

* npm
  ```sh
  npm install
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DiegoOrtegaElRincon/projecto
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project
   ```sh
   npm start
   ```
